## Prototype V1 — Features
- ✅ Multi-role login: Student / Teacher / Admin
- ✅ 5 thematic units × 15 Tier-2 vocabulary words = 75 words total
- ✅ 4-round progressive game engine (Remember → Understand → Apply → Analyze)
- ✅ 3-tier AI hint system (Semantic → Morphological → Structural)
- ✅ Individual progress tracking via localStorage
- ✅ Mastery star system per word (0–3 stars)
- ✅ Teacher dashboard with class overview + CSV export
- ✅ Admin board with multi-school analytics
- ✅ Offline-capable (no internet required for core gameplay)
- ✅ Responsive design (desktop, tablet, mobile)
- ✅ Resume facility (continues from last saved point on any device)

- **Cycle 2:** Teacher usability evaluation
